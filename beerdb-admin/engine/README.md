## Demo

Try the `beer.db` web admin demo running
on Heroku [`prost.herokuapp.com`](http://prost.herokuapp.com).


## Usage

### Rails

In your routes (`config/routes.rb`) mount the beer.db.admin app; add the line:

    mount BeerDbAdmin::Engine, :at => '/db'
